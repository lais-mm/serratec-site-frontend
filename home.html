Projeto Serratec

O programa visa aumentar a oferta de profissionais qualificados em tecnologia, dando oportunidade a jovens e adultos, e atrair novas empresas à Região Serrana do Estado do Rio de Janeiro.

Lógica de programação

É um paradigma de programação que faz uso da lógica matemática. Sendo a lógica utilizada a partir de 3 aplicaçoes web do Front-end: HTML5, JS (JavaScript) e CSS3.

Banco de dados

Conceitos de banco de dados, sistema de banco de dados, SGBD, modelo de banco de dados, modelo relacional, tipos de dados, constraints, normalização. O banco de dados utilizado em aula foi o SQL Server.

Java

Linguagem de programação orientada a objetos, voltada ao backend. Criação de projetos no Eclipse com Maven e Springboot

Frontend

Criação de páginas em HTML5, CSS3, Bootstrap Básico, JavaScript ES6 Básico.
